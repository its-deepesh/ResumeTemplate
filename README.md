# 📄 Resume Website

A modern and responsive resume website built using **HTML5** and **CSS3**. This project showcases my education, technical skills, projects, achievements, and contact information in a clean, professional layout inspired by a modern resume design.

---

## 🚀 Live Preview

🌐 **[View Resume Website](https://its-deepesh.github.io/ResumeTemplate/)**

---

## ✨ Features

- Semantic HTML5 structure
- External CSS styling
- Responsive three-column layout
- Professional resume design
- Profile picture section
- Contact information with icons
- Projects with GitHub links
- Skills and language proficiency
- Education section
- Clean and modern UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Font Awesome
